## 📱 **نظام WhatsApp المتكامل**

### 🔗 **الاتصال بـ WhatsApp**

1. **انتقل لصفحة WhatsApp:**
   ```
   النظام ← WhatsApp ← بدء الاتصال
   ```

2. **مسح QR Code:**
   - افتح WhatsApp على الموبايل
   - اذهب إلى الإعدادات ← الأجهزة المرتبطة
   - انقر "ربط جهاز" وامسح الكود

3. **التأكد من الاتصال:**
   - ستظهر رسالة "متصل ✅"
   - يمكنك الآن إرسال الرسائل

### 📤 **إرسال الرسائل**

#### **الرسائل الفردية:**
- اختر الموظف أو العميل
- اكتب الرسالة
- انقر "إرسال"

#### **الرسائل الجماعية:**
- اختر "رسالة جماعية"
- حدد المجموعة المطلوبة (موظفين/عملاء)
- اكتب الرسالة وأرسل

#### **الرسائل التلقائية:**
- رسائل الرواتب الشهرية
- تنبيهات المواعيد المهمة
- إشعارات الطلبات والموافقات
